## What I Build

Daniel's work spans a range of problem spaces, united by a preference for clean, maintainable code and declarative paradigms.

- **Modular Android apps with clean architecture** — Projects like *AlejoTaller* and *Deluxe* demonstrate a features + core modular layout, dependency injection with Koin, local persistence with Room, and a backend powered by Appwrite BaaS.
- **3D rendering experiments** — *Spatial* is a declarative 3D rendering library for Android, inspired by Jetpack Compose and powered by OpenGL ES 3.0. *SatExplorer* takes it further with real-time immersive visualisation of satellites and orbital paths.
- **On-device AI** — *Iris* is an experimental object-detection app running YOLO11s inference entirely on-device using CameraX — no server required.
- **Svelte web frontends** — *dash\_alejo\_taller* is a Svelte + TypeScript management dashboard that pairs with the AlejoTaller Android app. *compose\_svelted* brings Compose-style declarative layout, immutable modifiers, and navigation to Svelte.
- **Serverless & Cloudflare Workers** — From starter templates to Google OAuth flows and Telegram notification bots, Daniel builds lightweight, composable backend functions deployable to Cloudflare Workers and Appwrite Functions.

<Note>
  Several projects are intentionally cross-disciplinary — for example, the architectural ideas behind Jetpack Compose directly informed the design of **compose\_svelted** and the **Spatial** rendering library.
</Note>
